## Section 3: Instructions for reproducing your results. 
To reproduce our results, first download the COVID dataset from the Data Link file in the data folder of the repository. Save the dataset locally, and make sure the file name matches the one used in the script. Next, place that data file in the same directory as the Jupyter Notebook located in the scripts folder. Then open the script or notebook and run all code cells in order from top to bottom. When the script finishes running, it will produce the same results shown in our project.
